# 🔄 task-manager スキルの更新手順（Windows）

task-manager のスキル（Claude への指示ファイル）を最新版に入れ替える手順です。
所要時間は約 5〜10 分。難しい操作はありません。フォルダにファイルをコピーするだけです。

> ⚠️ この作業で、Notion の案件データが消えることはありません。触るのは「Claude への指示ファイル」だけです。

---

## ステップ 1: 最新版をダウンロード

1. 下のリンクをブラウザで開く
   👉 https://github.com/taiga-tsutsumaru/task-manager/releases/latest

2. ページの下の方にある「**Assets**」という項目を開く（すでに開いていることもあります）

3. その中の「**Source code (zip)**」をクリック
   → `task-manager-1.0.10.zip`（数字はその時の最新版）のようなファイルがダウンロードされます
   ※「start-guide-ja.pdf」や「Source code (tar.gz)」ではなく、必ず **Source code (zip)** を選んでください

---

## ステップ 2: ダウンロードしたファイルを解凍

1. ダウンロードした zip ファイル（通常は「ダウンロード」フォルダにあります）を右クリック

2. 「**すべて展開**」をクリック → 「**展開**」を押す

3. `task-manager-1.0.10` という名前のフォルダが出てきます

4. そのフォルダを開くと、中に「**skills**」というフォルダがあります。
   その skills フォルダを開くと、下記の 7 個のファイルが入っています：
   - tm-setup.md
   - tm-sync.md
   - tm-pending.md
   - tm-report.md
   - tm-add-talent.md
   - tm-add-client.md
   - tm-help.md

   👉 この 7 個を、あとでコピーします。このウィンドウは開いたままにしておいてください。

---

## ステップ 3: コピー先のフォルダを開く

ここが一番大事です。**Claude が指示ファイルを読みに行く専用フォルダ**を開きます。

1. キーボードで「**Windows キー**」を押しながら「**R**」を押す
   → 小さな「ファイル名を指定して実行」という窓が出ます

2. その窓に、下の文字を**そのままコピーして貼り付け**て、Enter を押す
   ```
   %APPDATA%\Claude\skills
   ```

3. すると「skills」という名前のフォルダが開きます。
   この中に、**今使っている（古い）tm- で始まるファイルが 7 個**入っているはずです。

> 💡 もし「フォルダが見つかりません」と出た場合：
> Claude Desktop を開いて、設定 → Skills の画面から「スキルフォルダを開く」に相当するボタンを探してください。そこから開いたフォルダが正しい場所です。分からなければ連絡ください。

---

## ステップ 4: 新しいファイルで上書き

1. **ステップ 2 で開いた** `task-manager-1.0.10\skills` フォルダの中の 7 個のファイルを、
   すべて選択します（Ctrl + A で全選択できます）

2. 選択したファイルを、**ステップ 3 で開いた** `%APPDATA%\Claude\skills` フォルダへ
   ドラッグ＆ドロップ（マウスでつまんで移動）します

3. 「**ファイルを置き換えますか？**」と聞かれたら、
   → 「**ファイルを置き換える**」（またはコピーして置き換え）を選ぶ

   ※ 同じ名前のファイルなので、古いものが新しいものに上書きされます。これで正解です。

---

## ステップ 5: Claude Desktop を再起動

1. Claude Desktop を一度完全に終了します
   - 画面右下のタスクトレイ（時計の近くの小さいアイコンの集まり）にある Claude のアイコンを右クリック → 「終了」

2. もう一度 Claude Desktop を起動します

---

## ステップ 6: 更新できたか確認

Claude Desktop のチャット欄に、下記を入力して送信してください：

```
/tm-help
```

スキルの一覧が表示されれば成功です。これで最新版のスキルに入れ替わりました。

---

## この後にやること

スキルの入れ替えが終わったら、**Notion 側のレイアウトを新しくする作業**が別途あります。
そちらは別のメッセージでお送りする「適用プロンプト」を Claude に貼り付けるだけです（先輩の手作業はほぼありません）。

---

## うまくいかないとき

- フォルダが見つからない / ファイルの場所が分からない
- 「置き換える」の選択肢が出ない
- /tm-help を送っても反応がない

→ そのまま連絡ください。画面を見ながら一緒に進めます。
